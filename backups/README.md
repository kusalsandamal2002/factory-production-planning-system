# Backups

`code_backups/` contains historical source snapshots retained for reference.
They are not imported by the running application.

`code_backups/local_snapshots/` contains local, generated backup directories.
That folder is ignored by Git and must not be treated as application source.

`code_backups/legacy_minute_scheduler/` contains the archived prototype
minute-scheduler services and UI pages. These files are retained for historical
reference only and are not imported by the current application.
