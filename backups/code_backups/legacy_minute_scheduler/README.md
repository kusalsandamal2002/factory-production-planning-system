# Legacy Minute Scheduler Archive

This directory contains the retired prototype scheduler and its dependent UI
modules. The current application does not import these files.

Archived content includes:

- minute-slot scheduling and break allocation;
- manager-priority schedule rebuilding;
- tire-type receive-date order preview and confirmation;
- shipment tabs tied to the retired order model;
- curing-time and tire-type master pages;
- legacy tire movement stock and production-entry pages.

These files are retained only for historical reference. Do not restore them to
`app/` without an approved migration plan and verified cycle-time source data.
